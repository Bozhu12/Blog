# Linux搭建

# 安装 JDK

  1. 上传JDK压缩文件，放置 根目录创建的 mysoft 文件夹

  2. 解压 (解压至当前目录)

    tar -xzvf {file}
    1

  3. 测试jdk安装。进入jdk的bin目录进行验证 (返回版本号表示成功)

    ./java -version
    1

    注意：在jdk的bin目录下执行java命令，前面要有 . 符号

  4. 配置环境变量 ，添加变量 JAVA_HOME、PATH

    vim /etc/profile
### 在最下面添加 两个变量 注意jdk的路径
export JAVA_HOME=/mysoft/jdk1.8.0_11
export PATH=$PATH:$JAVA_HOME/bin
    1
    2
    3
    4

  5. 使环境变量生效

    source /etc/profile
    1
    2

  6. 测试环境配置 (返回预期的bin目录即可)

    echo $PATH
    1

  7. 在任意路径测试java命令 (返回版本号代表成功)

    java -version
    1

# 安装 MySQL

  1. 检测是否自带安装MySQL

    yum list installed | grep mysql
    1

  2. 删除系统自带的sql及其依赖 (以防后面冲突)

    yum -y remove mysql-libs.x86_64
    1

  3. 添加rpm源

    wget --no-check-certificate dev.mysql.com/get/mysql-community-release-el6-5.noarch.rpm
    1

  4. 安装第一步下载的rpm文件 (出现许可输入y 即可

    yum install mysql-community-release-el6-5.noarch.rpm
    1

  5. 查看MySQL是否下载 ，列出所有yum repo文件

    yum repolist enabled | grep mysql
    1

  6. 使用yum安装MySQL （输入3次 y

    yum install mysql-community-server
    1

  7. 启动MySQL服务

    service mysqld start
    1

  8. 查看是否自动启动并启动

    # 查看
chkconfig --list | grep mysqld
# 自动启动
chkconfig mysqld on
    1
    2
    3
    4

    看到3、4、5状态为开或者为 on 则表示成功

  9. 修改字符集为UTF-8

    vim /etc/my.cnf
### 1. 在 [mysqld] 下面添加：
character-set-server=utf8
### 2. 在末尾 [client] 添加：(client没有自行添加)
default-character-set=utf8
    1
    2
    3
    4
    5

  10. 重启服务

    service mysqld restart
    1

  11. 修改默认配置

    mysql_secure_installation
    1

    此步骤可设置密码，root用户密码可自行设置。此处输入内容依次为：y、y、n、y、y

  12. 授权远程登录，通过改表实现

    # 上号
mysql -uroot -p
# 选中库
>use mysql;
# 修改表数据
>update user set host = '%' where user = 'root' and host='localhost';
# 允许外网访问
>grant all privileges on *.* to 'root'@'%' identified by 'PANZER' with grant option;
# 刷新MySQL的系统权限相关表
>flush privileges;
    1

  • 授权问题
  • 是否刷新权限表
  • 是否关闭防火墙 或 允许访问

# 安装 Tomcat

  1. 安装JDK

  2. 上传Tomcat压缩包到 根目录的 mysoft 文件夹

  3. 解压

    tar -xzvf {file}
    1

  4. 进入bin目录启动Tomcat

    sh startup.sh
    1

  5. 关闭防火墙

    # 查看防火墙状态
systemctl status firewalld
# 启动防火墙
systemctl start firewalld
# 关闭防火墙
systemctl stop firewalld
    1
    2
    3
    4
    5
    6

  6. 测试访问，

    # 查看IP
ifconfig
# 浏览器访问（看见熟系的页面）
{IP}:8080
    1
    2
    3

# 安装 Zookeeper

  1. 安装JDK

  2. 上传Zookeeper压缩包到根目录的 mysoft 文件夹

  3. 解压 解压至根路径usr

    tar -zxvf {file} -C /usr/
    1

  4. 进入 Zookeeper目录 创建 data 目录

    mkdir data
    1

  5. 编辑配置 ，进入confg目录 拷贝 zoo_sample.cfg 为 zoo.cfg (将默认配置更改并应用)

    # 拷贝应用的配置
cp zoo_sample.cfg zoo.cfg
# 编辑配置
vim zoo.cfg
### 更改 dataDir属性 路径 file为解压的文件目录
dataDir=/usr/{file}/data
    1
    2
    3
    4
    5
    6

  6. Zookeeper 启动&停止&查看

    # Zookeeper 启动
./zkServer.sh start
# Zookeeper 停止
./zkServer.sh stop
# Zookeeper 查看
./zkServer.sh status
    1
    2
    3
    4
    5
    6

    需要进入Zookeeper的bin目录且是root权限进行操作
